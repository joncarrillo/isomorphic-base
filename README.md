# isomorphic-base

A custom node/express + react isomorphic web app template.

## Getting Started

These instructions will get you a copy of the project up and running on your local machine for development and testing purposes.

### Prerequisites

* Install [NVM](https://github.com/nvm-sh/nvm)
* Install Node 14

```
nvm install v14
```

```
nvm use
```

### Installing

```
npm install
```

## Running linters

All.

```
npm run lint
```

JS (eslint).

```
npm run lint:js
```

CSS (stylelint).

```
npm run lint:css
```

## Running the tests

With Jest and RTL.

```
npm run test
```

### Running the app

```
npm run start
```
